===========================
How to Use 0-add_integer.py
===========================

This module defines a division of matrix 

Usage
=====

::

    >>> matrix_divided = __import__('2-matrix_divided').matrix_divided
    >>> matrix_divided([[1, 2, 3], [4, 5, 6]], 3)
    [[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]

1- Case matrix is not list of lists
2- Case element in lists are not [int, float]
3- Case matrixes don't have same size
4- Case div not a number
5- Case div is number but equal to 0
